# Agent Self-Check Instructions

Please execute the following tests to determine your capabilities and generate an agent profile.

### Tests:
1. **File I/O**: Create a temp file, write "ok", read it, delete it.
2. **Shell**: Run `echo hello` and check output.
3. **Background**: Run `sleep 2 &` and check if it returns immediately.
4. **MCP**: Check if `mcporter` command exists.
5. **Context**: Ask yourself "What is my context window size?".

### Output:
Based on the results, fill out `agents/<agent-name>.md` using the template in this directory.
